#

Question Asked


**Business Question:**

What is the current resource allocation across departments?

---

#

SQL Query


```sql

SELECT
department,
COUNT(*) as headcount,
SUM(budget) as budget,
ROUND(AVG(utilization_rate), 2) as utilization
FROM employees
JOIN department_budgets USING (department_id)
GROUP BY department
ORDER BY budget DESC

```
